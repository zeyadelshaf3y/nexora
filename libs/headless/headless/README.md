# @nexora-ui/headless

Meta package that installs all published headless libraries from Nexora.

## Install all headless packages

```bash
npm i @nexora-ui/headless
```

## Install individual packages

If you want only one primitive, install it directly:

```bash
npm i @nexora-ui/mention
```

## Included packages

- `@nexora-ui/core`
- `@nexora-ui/overlay`
- `@nexora-ui/interactions`
- `@nexora-ui/listbox-cdk`
- `@nexora-ui/listbox`
- `@nexora-ui/dropdown`
- `@nexora-ui/popover`
- `@nexora-ui/menu`
- `@nexora-ui/tooltip`
- `@nexora-ui/snackbar`
- `@nexora-ui/select`
- `@nexora-ui/combobox`
- `@nexora-ui/mention`
