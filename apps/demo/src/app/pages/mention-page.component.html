Mention

Headless mention primitive: type a trigger (@, #, /) to open a suggestion panel at the caret. Select with Enter, Tab, or click. The library ships zero opinionated styles — all visual styling is provided by the demo via nxrMentionEditorClass, nxrMentionChipClass, and CSS targeting [data-empty] for placeholders.

Basic @ users

Single trigger @, sync getItems, openOnTrigger: true, selectOnTab: true. Placeholder via CSS [data-empty]::before. Editor styling via nxrMentionEditorClass.
