Listbox

Headless listbox primitive: item-based options, selection and action modes, keyboard (arrows, Home/End, Enter, Space, typeahead), ARIA. The directive owns host focusability, keyboard, and all ARIA attributes automatically.

Single-select (object options, accessors, disabled option):
@for (item of fruits; track item.id) {
{{ item.name }}
}

Selected: {{ singleValue() ? singleValue()!.name : '—' }}

Multi-select (Space toggles, Enter activates):
@for (item of fruits; track item.id) {
{{ item.name }}
}

Selected: {{ getMultiSelectedNames() }}

Action mode (menu role, no value — optionActivated only):
@for (action of menuActions; track action.id) {
{{ action.label }}
}

Last activated: {{ lastActivated() ? (lastActivated() | json) : '—' }}

}
